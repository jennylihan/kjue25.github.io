Types of Telekinisis

In this class you will learn about the different types of telekennisis. These include being able to control objects, sensing others with telekenisis, and stoping the use of telekenisis in an area. You will learn how to use these different types of telekenisis.

Telekenisis Practice

In this class you will learn how to control objects with telekenisis as well as stopping the use of telekenisis and sensing telekenises. You will gradually advance from lifting papers and crayons to heavyer things like chairs and tables.

Nap Time

Telekenises uses up a lot of energy so we have a class specificly designed to teach students how to fall asleep faster. We will have half an hour every day of naping.
